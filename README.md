📌 Project Title

Project Management Form using JsonPowerDB (JPDB)

📄 Description

This is a simple Project Management Web Application built using HTML, CSS, JavaScript, jQuery, and JsonPowerDB (JPDB).

The system allows users to:

Add new project records
Search existing projects using Project ID
Update project details
Store data directly in JsonPowerDB database (COLLEGE-DB → PROJECT-TABLE)

The Project ID acts as the primary key for all database operations.

⚙️ Scope of Functionalities
Insert new project records (Save)
Fetch existing records using Project ID
Update existing records
Form validation (no empty fields allowed)
Auto enable/disable form fields based on database state
Reset form to initial state
🧠 Benefits of using JsonPowerDB
⚡ Very fast NoSQL database (low latency)
🔥 Serverless architecture (no backend required)
📡 Easy REST API integration
🧩 Schema-free data storage
💻 Simple JavaScript-based CRUD operations
🚀 Reduces development time significantly
📦 Lightweight and easy to deploy
